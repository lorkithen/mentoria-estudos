# Object

Os objetos javascript são coleções de propriedades e métodos. Um método
é uma função que é membro de um objeto. A propriedade é um valor ou um conjunto
de valores que é membro de um objeto. 

## Propriedades e Métodos Expando

As propriedades e os métodos podem ser adicionados e removiso em tempo de execução.
Podem ter qualquer nome e ser itendificados por números. Se o nome da propriedade
ou do método for um identificador simples, ele poderá ser escrito depois do nome
do objeto com um ponto final. 

### Exemplo:
```javascript
myObj.name
myObj.age
myObj.getage
```
### Exemplo de uso:
```javascript
var myObj = new Object();
myObj.name = "Luccas";
myObj.age = 21;
```
