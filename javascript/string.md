# String

É um dado de cadeira de caracteres. É uma cadeia de zero ou mais caracteres
(letras, sinais de pontuação e dígitos). São usados para representar um texto
no Javascript. Você delimita os caracteres com aspas simples ou duplas. As duplas
podem estar nas cadeias com aspas simples e as aspas simples podem estar
nas cadeias delimitadas por aspas duplas.

### Ex:

```javascript
"Estou feliz hoje"
'"Disse o sábio..."'
```

O JavaScript não tem um tipo de dado para representar um único caractere.
Voce pode criar uma cadeia de caracteres que possui apenas um caractere.

Existe também os caracteres de escape. Servem para realizar em uma cadeia de texto
determinados controles como uma quebra de linha, etc. Estes caracteres de escape
se escrevem inicialmente com uma contra-barra("\") e em seguida o código do caractere.

### Ex:

Quebra de linha: \n 

Aspas simples: \' 

Aspas dupla: \" 

Tabulador: \t 

Enter: \r 

Avance de página: \f 

Retroceder espaço: \b
