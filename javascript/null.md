# Null

O tipo de dados null tem apenas um valor: **null**. Uma variavel que contem
null não possui nenhum número, cadeia de caracteres, boolean, matriz ou objeto.
O valor **null** atribuído te permite apagar o conteúdo de uma variável atribuindo
o **null**.

### Exemplo
```javascript
// foo não existe e jamais será inicializado.
> foo

// foo é conhecido e existe mas não aponta nenhum tipo ou valor
> var foo = null; foo
```
