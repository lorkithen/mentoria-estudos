# Array

Os objetos e arrays são utilizados de forma quase idêntificas. Array
é simplesmente um tipo especial de objeto. Ambos podem ter propriedades
e métodos como ditos.

Um Array é um mapa numericamente indexados de valores.  Permite o armazenamento
de vários valores em uma mesma referência.

### Exemplo:

```javascript
var myVariable = 
[1,'bob','steve',10];
```
Assim você chama cada membro do array: myVariable[0],myVariable[1], etc.
