# Boolean

Os tipos de dados boolean só podem ter dois dados. Eles são **true** e **false**. O valor
**verdadeira** (true) especifica se a condição é verdadeira ou não. Se a condição
é verdadeira então executo uma instrução, se for não executo outra. É assim
que é lido pela variável.

### Aqui vai um exemplo:

```javascript
y = (x == 2000);
```

Nesse exemplo o valor **X** é comparado ao número **2000**. Se a comparação for exatamente
essa, o resultado booliano será true, se a variável **X** for diferente, o valor 
será false.
