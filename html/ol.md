# **ol**

> - Representa uma lista de itens ordenados. São mostrados como uma contagem ordenadas
que podem ser números, letras, algarismos romanos ou até mesmo símbolos.
> - Não há uma limitação para as listas definidas com esse comando.

## Ex:
```html
<html>
	<head>
		<title></title>
	</head>

	<body>
		<ol> Listar ordenadas 
			<li> Banana </li>
			<li> Maçã </li>
			<li> Caqui </li>
		</ol>
	</body>
</html>
```
