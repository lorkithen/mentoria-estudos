# **div**

> - É um elemento utilizado para *agrupar outros elementos*.
> - Ele utiliza uma quebra de linha antes de sua abertura e outra após. Basicamente serve para montar o layout.
> - Pode ser utilizado como um *"divisor"* de seções.
> - Junto do CSS acaba sendo uma das tags mais **utilizadas**.

## Exemplo
```html
<html>
	<head>
		<title></title>
	</head>

	<body>
		<div>
			<p> Alguma coisa </p>
		</div>
		<div>
			<p> Alguma coisa 2 </p>
		</div>
	</body>
</html>

Podemos também acrescentar **style** nas div's.

<html>
	<head>
		<title></title>
	</head>

	<body>
		<div style="background-color:#000000; text-align:center">
			<p> Alguma coisa </p>
		</div>
		<div>
			<p> Alguma coisa 2 </p>
		</div style="border:3px;>
	</body>
</html>
```
