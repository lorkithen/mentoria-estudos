# **Heading Tags**

> - É uma maneira de mostrar para o buscador como nosso site está organizado.
> - São usadas na estrutura e organizaão da página.
> - Citam qual conteúdo é mais importante. Sendo H1 o mais importante e H6 de menos importância.
> - A tag h1 só pode ser utilizada uma vez.

## Ex:
```html
<html>
<head>
<title>Exemplo</title>
</head>

<body>
	<h1> Esse artigo é de extrema importância.</h1>
		<p> Uma descrição do artigo... </p>
	<h2> Esse artigo é o segundo mais importante</h2>
		<p> A descrição desse... </o>
</body>
</html>
```
