# **ul**

> - Representa uma lista de itens que **não** são ordenados.
> - São exibidos com um marcador que podem obter varias formas que podem ser mudados no CSS.

## Ex:
```html
<html>
	<head>
		<title></title>
	</head>

	<body>
		<ul> Listar ordenadas 
			<li> Banana </li>
			<li> Maçã </li>
			<li> Caqui </li>
		</ul>
	</body>
</html>
```
